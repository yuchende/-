异或：

- 相同为0，不同为1

- a ^ a  = 0

- a ^ 0 = a

  

  题目：只出现一次得数字

  ​	给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了	一次的元素。

  ​	说明：

  ​	你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

  ​	示例 1:

  ​	输入: [2,2,1]
  ​	输出: 1
  ​	示例 2:

  ​	输入: [4,1,2,1,2]
  ​	输出: 4

```java
思路：异或 

public int singleNumber(int[] nums) {
        int res = 0;
        for(int i=0;i<nums.length;i++){
            res = res ^ nums[i];
        }
        return res;
    }
```
